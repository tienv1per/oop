Câu hỏi:
  If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?

Trả lời: 
  Không: Vì phương thức khởi tạo trong Java khi tạo DVD bằng title hay category có cùng 1 chữ ký(do cả 2 đều là cùng kiểu dữ liệu String) nên sẽ không được.
